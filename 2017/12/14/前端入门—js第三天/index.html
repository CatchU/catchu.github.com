2.数组

2.1声明数组

  • 使用[]; 如：var arr1 = [1,2,3,4,5];
  • 使用Array关键字；如：var arr2 = new Array(1,3,5,7,9);

    2.2使用数组

    使用数组的方法是：数组名[索引值]。但是需要注意的是索引值是从0开始算的。如：arr1[1] = 2;

    2.3数组长度

    计算数组长度的方法是：数组名.length。例如：arr1.length = 5.

    2.4数组遍历

    数组的遍历使用for循环。
